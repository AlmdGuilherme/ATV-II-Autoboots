# AutoManager

## Guia de Instalação

### Pré-requisitos

* [Java 17](https://www.oracle.com/java/technologies/downloads/#java17) (Versão utilizada no projeto)
* [Maven](https://maven.apache.org/download.cgi)
* [MySQL Server](https://dev.mysql.com/downloads/mysql/)
* Insomnia ou Postman para testar as rotas

---

### 1. Em um terminal, clone o repositório:

```bash
git clone https://github.com/AlmdGuilherme/ATV-II-Autoboots.git
cd ATV-II-Autoboots
```

---
### 2. Compile e instale as dependências e execute a aplicação::
```bash
mvn clean install
mvn spring-boot:run
```
